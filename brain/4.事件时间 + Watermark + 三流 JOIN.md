# 七、核心实现（Join Function）

---

## 1️⃣ Order + User

```java
public class OrderUserJoinFunction
        extends BroadcastProcessFunction<Order, User, WideOrder> {

    private final MapStateDescriptor<Long, User> stateDesc =
            new MapStateDescriptor<>("user-state", Long.class, User.class);

    @Override
    public void processElement(
            Order order,
            ReadOnlyContext ctx,
            Collector<WideOrder> out) throws Exception {

        User user = ctx.getBroadcastState(stateDesc).get(order.getUserId());

        WideOrder wide = new WideOrder();
        wide.setOrderId(order.getOrderId());
        wide.setUserId(order.getUserId());
        wide.setAmount(order.getAmount());

        if (user != null) {
            wide.setLevel(user.getLevel());
        }

        out.collect(wide);
    }

    @Override
    public void processBroadcastElement(
            User user,
            Context ctx,
            Collector<WideOrder> out) throws Exception {

        ctx.getBroadcastState(stateDesc)
                .put(user.getUserId(), user);
    }
}
```

---

## 2️⃣ 再 JOIN Member

```java
public class OrderMemberJoinFunction
        extends BroadcastProcessFunction<WideOrder, Member, WideOrder> {

    private final MapStateDescriptor<Long, Member> stateDesc =
            new MapStateDescriptor<>("member-state", Long.class, Member.class);

    @Override
    public void processElement(
            WideOrder order,
            ReadOnlyContext ctx,
            Collector<WideOrder> out) throws Exception {

        Member member = ctx.getBroadcastState(stateDesc)
                .get(order.getUserId());

        if (member != null) {
            order.setMember(member.getIsMember());
        }

        out.collect(order);
    }

    @Override
    public void processBroadcastElement(
            Member member,
            Context ctx,
            Collector<WideOrder> out) throws Exception {

        ctx.getBroadcastState(stateDesc)
                .put(member.getUserId(), member);
    }
}
```
